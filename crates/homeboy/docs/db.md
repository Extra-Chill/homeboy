# db

Database operations for projects.

```bash
homeboy db tables <project-id> [args...]
homeboy db describe <project-id> <table>
homeboy db query <project-id> <sql>
homeboy db delete-row <project-id> <table> <where-clause>
homeboy db drop-table <project-id> <table>
homeboy db tunnel <project-id>
```

Use `homeboy help db` for the authoritative flag list.
