# file

Remote file operations for projects.

```bash
homeboy file list <project-id> <path>
homeboy file read <project-id> <path>
homeboy file write <project-id> <path>
homeboy file delete <project-id> <path>
homeboy file rename <project-id> <old-path> <new-path>
```

Use `homeboy help file` for the authoritative flag list.
